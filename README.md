# Family Achievements

Telegram-бот + HTTP API для семейной системы достижений. Участники семьи получают ачивки за реальные дела — спорт, учёбу, творчество, быт. Администраторы подтверждают заявки, прогресс отображается в боте и через API для веб-клиента.

---

## Функциональность

### Достижения

- **Дерево зависимостей** — у каждой ачивки могут быть предварительные условия (prerequisites): нельзя получить «Бегун», не получив «Первая пробежка»
- **Редкость** — пять уровней: Common / Uncommon / Rare / Epic / Legendary, каждый со своим цветовым эмодзи
- **Повторяемые ачивки** — можно получать несколько раз (с уровнями), у каждой опциональный максимальный уровень
- **Очки** — каждая ачивка приносит баллы
- **Иконки** — эмодзи-иконка для каждой ачивки
- **Категории** — ачивки сгруппированы по категориям (Здоровье, Образование, Творчество, Быт, Общение и др.)
- **Статусы** — `LOCKED` (заблокирована), `AVAILABLE` (доступна), `ACHIEVED` (получена)

### Бот — групповой чат

| Команда | Описание |
|---|---|
| `/register` | Зарегистрировать группу и стать администратором (только для Telegram-администраторов чата) |
| `/join` | Вступить в группу как участник |
| `/members` | Показать список участников группы с разбивкой по ролям |
| `/achievements` | Каталог всех достижений, сгруппированный по категориям и отсортированный по редкости |
| `/help` | Список всех команд |
| `/ping` | Проверить работу бота |

### Бот — личный чат

После `/start` открывается главное меню с разделами:

| Раздел | Описание |
|---|---|
| ✅ Доступные | Ачивки, которые можно получить прямо сейчас, сгруппированные по категориям |
| 🏆 Полученные | Подтверждённые достижения пользователя |
| 🔒 Заблокированные | Ачивки, у которых ещё не выполнены prerequisite-условия |
| 📩 Мои заявки | История поданных заявок с их статусами |
| 🧾 Заявки на подтверждение | Панель администратора — очередь входящих заявок |
| 🔄 Сменить группу | Переключиться между группами (если состоишь в нескольких) |
| 🚪 Покинуть группу | Выйти из текущей группы |

### Заявки на ачивки

1. Участник открывает ачивку и нажимает **«Заявиться»**
2. Бот просит описать доказательство (текст, фото, ссылка) или разрешает пропустить через `/skip`
3. Заявка попадает в очередь администратора
4. Администратор подтверждает (✅) или отклоняет (❌) с опциональным комментарием

### Уведомления

- **Администратору в личку** при новой заявке — имя заявителя, ачивка с иконкой и редкостью, превью доказательства
- **Участнику в личку** при подтверждении или отклонении заявки
- **В общий чат группы** при подтверждении — публичное поздравление в формате:

  ```text
  🏆 @username получил(а) ачивку «Дух марафонца» 🥇 (RARE) — уровень 3/10!
  ```

### HTTP API

REST API для веб-клиента (дерево ачивок, статусы, категории).

## Быстрый старт

### Локально

```bash
# 1. Создать виртуальное окружение (Python 3.12)
python3.12 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

# 2. Настроить переменные окружения
cp .env.example .env
# Вписать BOT_TOKEN в .env

# 3. Применить миграции
alembic upgrade head

# 4. Загрузить стартовые достижения
python -m app.seed.import_achievements

# 5. Запустить
python main.py
```

### Docker

```bash
cp .env.example .env
# Вписать BOT_TOKEN в .env

docker compose up --build
```

## Сценарий первого запуска

```
1. Создать бота через @BotFather, получить BOT_TOKEN
2. Добавить бота в семейный групповой чат
3. Администратор → /register в группе
4. Каждый участник → /join в группе
5. Каждый участник → /start боту в личку
6. python -m app.seed.import_achievements  (загрузить 15 готовых ачивок)
```

## API

**Базовый URL:** `http://localhost:8000`

**Документация:** `http://localhost:8000/docs`

### Эндпоинты

| Метод | Путь | Описание |
| --- | --- | --- |
| `GET` | `/api/groups/{group_id}/users/{user_id}/tree` | Полное дерево ачивок пользователя с зависимостями и статусами |
| `GET` | `/api/groups/{group_id}/users/{user_id}/achievements` | Список ачивок с фильтром `?status=AVAILABLE\|ACHIEVED\|LOCKED` |
| `GET` | `/api/groups/{group_id}/categories` | Все категории достижений |

### Статусы достижений

| Статус | Описание |
|---|---|
| `LOCKED` | Не выполнены prerequisites |
| `AVAILABLE` | Можно подать заявку |
| `ACHIEVED` | Подтверждено администратором |

## Стартовые достижения (seed)

15 ачивок в 5 категориях с деревом зависимостей:

| Категория | Ачивки |
|---|---|
| Здоровье | Первая пробежка → Бегун → Дух марафонца ♻️, Жаворонок ♻️ |
| Образование | Книжный червь ♻️, Полиглот, Дипломированный ♻️ |
| Творчество | Первый рисунок → Творец ♻️, Музыкант |
| Быт | Шеф-повар ♻️, Чистюля ♻️, Мастер на все руки ♻️ |
| Общение | Семейный день ♻️, Командный игрок ♻️, 👑 Семейная легенда |

♻️ — повторяемая ачивка

Для загрузки: `python -m app.seed.import_achievements`
Для кастомных ачивок: `python -m app.seed.import_achievements --file path/to/custom.yaml`
